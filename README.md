# Odin Recipes 

## Overview

- Recipe Website made using only basic HTML. My first website.
- Made as part of The Odin Project curriculum. 
- This site contains multiple recipes for traditional Dominican dishes.

---

## Objectives

With this project I have learned how to correctly use basic HTML and css, preparing myself to continue learning.

---

## Skills Demonstrated

- Linking multiple HTML pages 
- Use of lists (ol/ul)
- Use of headings to structure pages 
- File organization
- use of div elemets for page structure
- use of class selectors to applay color, background color, font, font size and manage boxes


## Technologies Used

- Desktop PC running Arch Linux (CachyOS)
- VS Code 
- ChatGPT (for generating recipe images, descriptions, steps, and ingredient lists)
